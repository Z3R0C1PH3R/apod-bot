# APOD Bot

Automated daily pipeline that turns NASA's [Astronomy Picture of the Day](https://apod.nasa.gov/apod/) into a vertical short, narrates it with Azure Neural TTS, and publishes it to **YouTube Shorts** and **Instagram Reels**.

It runs serverlessly on **Azure Container Apps Jobs** (scales to zero between runs) on a daily schedule.

---

## How it works

```
NASA APOD API ──> download image
       │
       ▼
Azure Speech (TTS) ──> audio.mp3 + word-timed audio.srt
       │
       ▼
ffmpeg ──> 1080x1920 short (blurred-bg intro + slow pan + burned-in subtitles)
       │
       ├──> YouTube Data API v3  (Shorts upload)
       └──> Azure Blob Storage (public URL) ──> Instagram Graph API (Reels)
```

| Stage | Module |
|-------|--------|
| Fetch APOD + image (with retries) | `img.py` |
| Text-to-speech + SRT subtitles | `azure_tts.py` |
| Video assembly (ffmpeg) | `main.py` (`process_video`) |
| YouTube upload (resumable, retrying) | `upload.py`, `Google.py` |
| Public hosting for Instagram | `blob_upload.py` |
| Instagram Reels publish | `insta.py` |
| Logging + retry helpers | `utils.py` |

## Requirements

- Python 3.12+
- `ffmpeg` on the PATH
- An Azure subscription (Speech + Blob Storage + Container Apps)
- A Google Cloud OAuth client for the YouTube Data API
- (Optional) An Instagram Business/Creator account linked to a Facebook Page

Install Python dependencies:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Configuration

All configuration is via environment variables (see `.env.example`):

| Variable | Purpose |
|----------|---------|
| `NASA_API_KEY` | NASA APOD API key (https://api.nasa.gov/) |
| `AZURE_SPEECH_KEY` / `AZURE_SPEECH_REGION` | Azure Speech resource |
| `AZURE_SPEECH_VOICE` | Voice (default `en-US-Andrew:DragonHDLatestNeural`) |
| `AZURE_SPEECH_STYLE` | Express-as style (multilingual voices only) |
| `AZURE_SPEECH_RATE` | Speech rate, e.g. `+12%` |
| `AZURE_STORAGE_CONNECTION_STRING` / `AZURE_STORAGE_CONTAINER` | Blob hosting for Instagram |
| `INSTAGRAM_ACCESS_TOKEN` / `INSTAGRAM_USER_ID` | Instagram publishing (optional) |
| `HEADLESS` | `1` in containers; fail fast instead of opening a browser for OAuth |
| `LOG_LEVEL` | `INFO` (default), `DEBUG`, etc. |

### Secrets (not in the repo)

These are git-ignored and must be provided locally / via Azure secrets:

- `client_secret_*.json` — Google OAuth client (download from Google Cloud Console)
- `token_youtube_v3.pickle` — generated by the re-auth step below

## Local usage

1. Authenticate YouTube once (opens a browser):

   ```bash
   python3 scripts/reauth_youtube.py
   ```

2. Run the pipeline (today's APOD, or a specific date):

   ```bash
   export NASA_API_KEY=...
   export AZURE_SPEECH_KEY=...
   export AZURE_SPEECH_REGION=eastus
   python3 main.py            # today
   python3 main.py 2026-06-20 # specific date
   ```

The bot exits `0` on success, `1` if it fails before publishing, and `2` if a publish step failed.

### Voice demo

Compare the recommended voices using the real APOD narration text:

```bash
python3 scripts/voice_demo.py 2026-06-20        # short recommended list
python3 scripts/voice_demo.py 2026-06-20 --all  # every en-US neural voice
```

Open `voice-demo/index.html` to listen.

## Deploy to Azure

The deploy script provisions everything (resource group, Speech S0, Blob Storage,
Container Registry, Container Apps environment) and creates/updates the scheduled job:

```bash
export NASA_API_KEY=...
export INSTAGRAM_ACCESS_TOKEN=...   # optional
./deploy/azure_setup.sh
```

The job is scheduled daily (cron `41 6 * * *` = **12:11 PM IST**). Override with `AZURE_CRON`.

```bash
# Run once now
az containerapp job start -g youtube-apod-rg -n youtube-apod-daily

# Tail logs
az containerapp job logs show -g youtube-apod-rg -n youtube-apod-daily \
  --container youtube-apod-daily --follow
```

## Notes

- **Reliability:** network calls (NASA, Azure TTS, Blob, Instagram, YouTube) retry with
  exponential backoff. ffmpeg intermediates are written to a temp dir and cleaned up.
- **Instagram token** is long-lived (~60 days). Refresh via the Facebook Graph API
  using your app ID/secret when it expires, then redeploy.
- The earlier hand-written AWS Polly / OpenAI / Piper TTS modules were removed in favor
  of Azure TTS.
